## Overview

This repository hosts a static, mobile-friendly HTML interface for quick access to IT & dev related news, blogs, communities and more.

* **News & Aggregators:** Hacker News, Heise, Golem, TLDR...
* **Engineering & Architecture:** InfoQ, Martin Fowler, The New Stack...
* **Security:** BleepingComputer, Krebs on Security, Kuketz...
* **Reference:** Stack Overflow, Linux Kernel (LWN), Phoronix...
